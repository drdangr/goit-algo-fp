# Завдання 2 — Фрактал «Дерево Піфагора»

## Опис

У цьому завданні реалізовано побудову фрактала **«Дерево Піфагора»**
з використанням **рекурсії** та модуля `turtle`.

Програма візуалізує дерево шляхом рекурсивного малювання гілок,
кожна з яких утворює дві нові гілки меншої довжини.

---

## Реалізація

- Використано **рекурсивну функцію** для побудови фрактала.
- Реалізовано **базовий випадок рекурсії**.
- Візуалізація виконана за допомогою стандартного модуля `turtle`.
- Користувач має можливість **задати рівень рекурсії** через введення з клавіатури.
- Кут повороту гілок становить **45°**, що відповідає прикладу з конспекта.
- у файлі capture.png приклад з глибиною 8

---

## Використання

При запуску програми необхідно ввести бажаний рівень рекурсії.
Зі збільшенням рівня зростає деталізація фрактала.

---

## Висновок

Виконано відповідно до вимог:
- використано рекурсію;
- реалізовано візуалізацію фрактала;
- забезпечено можливість керування рівнем рекурсії.
