# goit-algo-fp

# Фінальний проєкт з курсу «Алгоритми»

Цей репозиторій містить фінальний проєкт з курсу «Алгоритми».
Кожне завдання реалізоване в окремій папці відповідно до умов курсу.

---

## Примітка

Кожне завдання є самодостатнім та містить власний файл `README.md`
з детальним описом реалізації та результатів.

## Структура проєкту

- `task_1_linked_list/`  
  **Однозв’язний список**  
  Реверсування списку, сортування та об’єднання відсортованих списків.  

- `task_2_pythagoras_tree/`  
  **Фрактал «Дерево Піфагора»**  
  Побудова фрактала з використанням рекурсії та візуалізацією через `turtle`.  

- `task_3_dijkstra/`  
  **Алгоритм Дейкстри**  
  Пошук найкоротших шляхів у зваженому графі з використанням бінарної купи.  

- `task_4_heap_visualization/`  
  **Візуалізація бінарної купи**  
  Побудова дерева на основі структури купи з використанням NetworkX та Matplotlib.  

- `task_5_tree_traversal/`  
  **Візуалізація обходу бінарного дерева (BFS та DFS)**  
  Реалізація обходів без рекурсії з використанням стеку та черги,
  з візуалізацією порядку обходу за допомогою кольорів.  

- `task_6_greedy_vs_dp/`  
  **Жадібний алгоритм та динамічне програмування**  
  Порівняння двох підходів для задачі вибору їжі з максимальною калорійністю
  в межах обмеженого бюджету.  

- `task_7_monte_carlo/`  
  **Метод Монте-Карло**  
  Моделювання кидків двох кубиків та порівняння емпіричних результатів
  з аналітичними ймовірностями.  

---

## Залежності

Необхідні сторонні бібліотеки перелічені у файлі `requirements.txt` у корені репозиторію.

Для встановлення залежностей виконайте:
pip install -r requirements.txt
