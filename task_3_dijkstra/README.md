# Завдання 3. Алгоритм Дейкстри

## Мета роботи
Реалізувати алгоритм Дейкстри для знаходження найкоротших шляхів у зваженому графі з використанням бінарної купи (піраміди).

---

## Умова задачі
Дано зважений граф з невід’ємними вагами ребер.  
Необхідно знайти найкоротші відстані від початкової вершини до всіх інших вершин графа.

---

## Реалізація

Граф представлено у вигляді словника суміжності, де:
- ключ — вершина графа;
- значення — словник сусідніх вершин та ваг ребер.

Для оптимізації вибору вершини з мінімальною поточною відстанню використано **бінарну купу** (`heapq`).

---

## Алгоритм

1. Ініціалізуються відстані до всіх вершин як нескінченність.
2. Відстань до стартової вершини встановлюється рівною 0.
3. Стартова вершина додається до бінарної купи.
4. Поки купа не порожня:
   - обирається вершина з мінімальною відстанню;
   - виконується релаксація всіх суміжних ребер;
   - оновлені відстані додаються до купи.
5. В результаті отримуються найкоротші відстані до всіх вершин графа.

---

## Результат

Алгоритм повертає словник з найкоротшими відстанями від початкової вершини до всіх інших вершин графа.
Для графа, описаного як

graph = {
    "A": {"B": 5, "C": 1},
    "B": {"A": 5, "C": 2, "D": 1},
    "C": {"A": 1, "B": 2, "D": 4},
    "D": {"B": 1, "C": 4}
}

Shortest distances from A:
A: 0
B: 3
C: 1
D: 4
---

## Висновок
Алгоритм Дейкстри з використанням бінарної купи дозволяє ефективно знаходити найкоротші шляхи у зважених графах з невід’ємними вагами ребер.  
Використання купи зменшує кількість операцій вибору мінімальної вершини та покращує продуктивність алгоритму.
