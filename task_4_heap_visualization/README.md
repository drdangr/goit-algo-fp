# Завдання 4. Візуалізація бінарної купи

## Мета роботи
Побудувати та візуалізувати бінарну купу у вигляді бінарного дерева, використовуючи наданий у конспекті код для візуалізації дерев.

---

## Реалізація

Для візуалізації використано такі кроки:

1. Вихідні значення взято з прикладу, наведеного у конспекті.
2. Дані приведено до структури бінарної купи за допомогою функції `heapq.heapify`.
3. Бінарну купу представлено у вигляді масиву.
4. На основі індексної структури купи (`2i + 1`, `2i + 2`) побудовано бінарне дерево.
5. Дерево візуалізовано з використанням бібліотек `networkx` та `matplotlib`.

---

## Результат
Отримано коректну візуалізацію бінарної купи у вигляді дерева, де:
- мінімальний елемент знаходиться у корені;
- структура дерева відповідає властивостям бінарної купи;
- форма дерева визначається індексною структурою купи.
- результат зафіксовано у файлі Figure_1.png
---

## Висновок
Завдання демонструє відповідність між масивним представленням бінарної купи та її деревоподібною структурою, а також можливість наочної візуалізації цієї структури.
